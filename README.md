# CoreELEC Temperature Monitor

## English
CoreELEC Temperature Monitor is a Kodi service add-on designed to monitor your device's temperature on CoreELEC and send instant alerts via Telegram when the temperature exceeds a defined threshold.

### Features
- Real-time temperature monitoring for CoreELEC devices.
- Automatic alerts via Telegram when temperature exceeds the limit.
- Test message option to verify settings.
- Adjustable settings for threshold, interval, and daily alert limit.

### Installation
1. Download the ZIP file of the add-on.
2. In Kodi, go to **Settings > Add-ons > Install from zip file**.
3. Select the downloaded ZIP file and install.

### Configuration
- Set your Telegram Chat ID in the add-on settings.
- Define temperature threshold and alert interval.

## العربية
**CoreELEC Temperature Monitor** هي إضافة لخدمة Kodi مصممة لمراقبة حرارة جهازك على نظام CoreELEC وإرسال تنبيهات فورية عبر تيليجرام عند تجاوز الحد المسموح.

### المزايا
- مراقبة حرارة الجهاز بشكل لحظي.
- إرسال تنبيهات تلقائية عبر تيليجرام عند تجاوز الحد.
- خيار إرسال رسالة اختبار للتأكد من الإعدادات.
- إعدادات مرنة للتحكم في الحد الأقصى والفاصل الزمني وعدد التنبيهات اليومية.

### طريقة التثبيت
1. قم بتحميل ملف الإضافة بصيغة ZIP.
2. في Kodi، انتقل إلى **الإعدادات > الإضافات > التثبيت من ملف ZIP**.
3. اختر الملف الذي تم تحميله وقم بالتثبيت.

### الإعدادات
- ضع معرف الدردشة (Chat ID) الخاص بك في إعدادات الإضافة.
- حدد الحد الأقصى للحرارة والفاصل الزمني للتنبيهات.
